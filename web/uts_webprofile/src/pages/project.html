Earthquake Early Warning System

This project integrates the USGS (United States Geological Survey) API to access real-time earthquake data for the Earthquake Early Warning System. The API provides live information on recent seismic events, including their location, magnitude, depth, and time of occurrence. By using this data, the system can quickly detect earthquakes, analyze the seismic activity, and issue alerts to users in affected areas. The integration of the USGS API ensures that the system relies on accurate, up-to-date information from a trusted source, making it more effective in delivering timely warnings and helping to reduce potential damage and injuries from earthquakes.

Running Calculator

This project is designed to help track your running workouts, especially if you don’t have a smartwatch or fitness tracker. While smartwatches can automatically calculate your pace, this tool is perfect for those who rely on basic equipment—just a running track and a watch with a timer or stopwatch. By inputting your time and distance, this tool quickly calculates your running pace, allowing you to monitor your progress and set new goals. It’s a simple yet effective solution for runners who want to keep track of their performance without investing in high-tech gadgets.

Travelsetia

This project was developed by a group of four people as part of our final assignment for the second semester. The application is inspired by popular booking platforms like Traveloka but is designed to be simpler and more focused, with a primary emphasis on flight booking functionality. Built using Java GUI and JavaFX, the application offers an intuitive and user-friendly interface that allows users to search for, view, and book flights efficiently. We integrated a MySQL database to handle and store flight data, user profiles, and booking information securely. This project not only provided us with practical experience in front-end and back-end development but also taught us how to work collaboratively on a complex application from planning to deployment.
